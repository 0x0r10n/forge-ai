# Changelog

All notable changes to Forge AI are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-05-26

### Changed
- Full project identity rebrand to Forge AI.
- CLI command standardized as `forge`.
- NPM package renamed to `forge-ai`.
- Distribution artifact standardized as `dist/forge.js`.
- Config and runtime storage namespace moved to `~/.forge`.
- Documentation, help text, onboarding, and terminal UI updated to Forge AI branding.
- Product tagline set to: **"Forge better code. Faster."**
- Product description set to: **"The open-source multi-agent coding agent that beats Cursor and Claude Code on price and control."**

### Notes
- Version remains `1.0.0` under the new branding.
- Functionality is unchanged outside naming, identity, and related command/documentation updates.
